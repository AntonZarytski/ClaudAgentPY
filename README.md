# Claude Chat Agent

Простой веб-чат с использованием Claude API.

## Установка
```bash
python3 -m venv venv
source venv/bin/activate  # Linux/Mac
pip install -r requirements.txt
```

## Настройка

Создайте файл `.env`:
```
ANTHROPIC_API_KEY=your_key_here
PORT=3000
```

## Запуск
```bash
python app.py
```

Откройте http://localhost:3000
